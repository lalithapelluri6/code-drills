The HTML4 Recommendation from 1997 shows an example of how Media Queries could be added in the future. In 2000, W3C started work on Media Queries and also on another scheme for supporting various devices: CC/PP. The two address the same problem, but CC/PP is server-centric, while Media Queries are browser-centric. The first public working draft for Media Queries was published in 2001, and the specification became a W3C Recommendation in 2012 after browsers added support.
